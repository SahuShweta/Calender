# 📅 React Calendar App

A simple, interactive calendar built using React. This project demonstrates date handling, state management, and dynamic UI rendering.

---

## 🚀 Features

- 📆 Display current month and year  
- ⬅️➡️ Navigate between months  
- 📌 Highlight today’s date  
- ✅ Select a specific date  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- React (Hooks: useState, useEffect)  
- JavaScript (ES6+)  
- CSS3  
- Vite
